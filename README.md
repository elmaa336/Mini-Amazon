# -Mini-Amazon
🛒 Mini-Amazon . Un backend e-commerce en C++ pour notre module de POO universitaire (équipe de 5). Il simule un marché numérique : gestion de produits via héritage/polymorphisme, moteur de tarification dynamique (taxes/remises) et système de recommandation par score de similarité. Code propre avec séparation stricte des .h et .cpp.
